### Переменные

Переменные в Python имею ==динамическую типизацию==:
 * Тип переменной определяется присвоенными данными
 * Одна и та же переменная может быть разного типа в разное время

```python
my_var = [1, 2, 3]
my_var = "string"
my_var = 'other_string'
my_var = "a"
my_var = 10
my_var = 10.2
```

> [!warning] Переменные хранят ссылки на объекты, а не сами объекты

--- 
### Объекты

В `Python` всё является объектом:
* Числа
* Строки
* Списки (и пр. структуры данных)

> [!note] Любой объект может быть присвоен переменной

В `Python` всё ведёт себя так как будто унаследовано от одного класса.

### Примеры списков

```Python
# Пустой список
list = []
# Список чисел
list = [1, 2, 3]
# Список символов
list = ['a', 'b', 'c']
# Список списков
list = [[1], [2, 3]]
# Список строк
list = ["str1", "str2"]
# Гетерогенный список
list = [1, 'a', "str1", [], [2, [3, 4]]]
```

---

### Поиск объекта внутри списка

Для поиска используется оператор `in`:
```python
vovels = ['a', 'e', 'i', 'o', 'u']
word = "Milliways"

for letter in word:
	if letter in vovels:
		print(letter)
		
for letter in word:
	if letter not in vovels:
		print(letter)
```

### Функции работы со списками

```python
list = [1, 2, 3, 4, 5]

# Длина списка - 5
len(list)
# Добавление в список 
list.append(4) # [1, 2, 3, 4, 5, 4]
# Удаление первого вхождения из списка 
list.remove(4) # [1, 2, 3, 5, 4]
# Удаление последнего объекта в списке и возврат
obj = list.pop() # [1, 2, 3, 5]
# Удаление бъекта по индексу и возврат
obj = list.pop(0) # [2, 3, 5]
# Добавление в список элементов другого списка
list.extend([10, 11]) # [2, 3, 5, 10, 11] 
# Вставка список элемента по индексу
# Элемент, стоявший по этому индексу сдвигается
list.insert(2, 4) # [2, 3, 4, 5, 10, 11] 
# Копирование списка
new_list = list.copy()
```

### Доступ по индексу

К элементам в списке можно доступиться как по положительному так и по отрицательному индексу (с конца):

```python
list = [1, 2, 3, 4, 5]
list[1] # 2
list[4] # 5
list[-4] # 2
list[-1] # 5
```

### Срезы 

Срезы - удобный инструмент выборки данных из списков:

```Python
list_name[start:stop:step]
```

> [!warning] Срез создаёт новый список но хранит ссылки на те же объекты, что были в исходном списке 

```python
>>> str = "Don't panic"
>>> l = list(str)
>>> l
['D', 'o', 'n', "'", 't', ' ', 'p', 'a', 'n', 'i', 'c']
>>> l[0:10:3]
['D', "'", 'p', 'i']
>>> l[3:]
["'", 't', ' ', 'p', 'a', 'n', 'i', 'c']
>>> l[3::2]
["'", ' ', 'a', 'i']
>>> l[::2]
['D', 'n', 't', 'p', 'n', 'c']
>>> l[-1::-2]
['c', 'n', 'p', 't', 'n', 'D']
>>> l[-1:-6:-3]
['c', 'a']
```

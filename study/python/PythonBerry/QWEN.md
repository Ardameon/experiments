# PythonBerry

Учебный проект по изучению Python.

## Структура проекта

| Директория | Описание |
|------------|----------|
| `c01_basics/` | Основы: переменные, условия, циклы |
| `c02_lists/` | Работа со списками, строками, срезами |
| `my_tests/` | Личные эксперименты и тесты |
| `my_tests/exercises_*/` | Тренировочные упражнения по датам |

Каждый каталог `c*` соответствует главе книги Пола Бэрри `Изучаем программирование на Python`
Внутри каждого такого каталога находятся примеры кода из книги.
Если такой каталог с соответствующим номером главы присуствует, то я уже изучил, либо изучаю эту главу.

## Запуск

```bash
# Примеры запуска
python3 c01_basics/odd.py
python3 c02_lists/marvin.py
```

## Окружение

- Python 3.10+
- ОС: Linux

## Стиль кода

- f-строки для форматирования
- Минимум импортов
- Простые и понятные решения

- базовые принципы языка Python
- import
- оператор if
- цикл for
- функция dir()
- получение справки - help()
- функция range()
- объект rand
- oбъект time
- базовая работа со списками
    * in
    * insert
    * append
    * remove
    * pop
    * взятие по индексам
    * срезы
    * копирование списков
    * генераторы списка вида [x for x in range(n, m)]
    * длина списка len()

    * превращение строки в список

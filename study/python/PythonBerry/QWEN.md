# PythonBerry

Учебный проект по изучению Python.

## Структура проекта

| Директория | Описание |
|------------|----------|
| `c01_basics/` | Основы: переменные, условия, циклы |
| `c02_lists/` | Работа со списками, строками, срезами |
| `my_tests/` | Личные эксперименты и тесты |
| `my_tests/exercises_*/` | Тренировочные упражнения по датам |

Каждый каталог `c*` соответствует главе книги Пола Бэрри `Изучаем программирование на Python`
Внутри каждого такого каталога находятся примеры кода из книги.
Если такой каталог с соответствующим номером главы присуствует, то я уже изучил, либо изучаю эту главу.

## Запуск

```bash
# Примеры запуска
python c01_basics/odd.py
python c02_lists/marvin.py
```

## Окружение

- Python 3.10+
- ОС: Linux
